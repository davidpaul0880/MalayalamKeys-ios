Setup

  1. Open Device Settings from Home screen
  2. Tap General -> Keyboard -> Keyboards -> New Keyboard
  3. Select "Easy Malayalam" Keyboard from THIRD-PARTY KEYBOARDS
  4. Optional: Tap "Easy Malayalam" Keyboard and tap Allow Full Access

Usage

  1. Tap and hold Globe icon on default keyboard and select മലയാളം


  2. Type in Malayalam in any text input field
  3. Tap Globe icon on "Easy Malayalam" Keyboard to switch back to other keyboards
