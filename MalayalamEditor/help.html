Shortcuts

ക, ഗ, ങ, ച, ജ, ഞ, ട, ഡ , ണ ,ത. ദ, ന , പ , ബ, മ , യ, ല, വ, ശ, സ, ള എന്നീ അക്ഷരങ്ങൾ ഇരട്ടിപ്പിക്കുന്നതിന് അതാതു keys double tap ചെയ്താൽ മതിയാകും.

  • ക double tap = ക്ക
  • ഗ double tap = ഗ്ഗ
  • ങ double tap = ങ്ങ
  • റ double tap = റ്റ
  • സ double tap = സ്സ
    • etc..

    ക, ച, ട, ത, പ എന്നീ വർഗ്ഗങ്ങളിൽ ഓരോന്നിന്റെയും അനുനാസികവും ഖരവും ചേർന്നു കൂട്ടക്ഷരമുണ്ടാക്കുന്നതിന് അതാതു keys അടുപ്പിച്ച് tap ചെയ്താൽ മതി.

  • ങ + ക = ങ്ക
  • ഞ + ച = ഞ്ച
  • ണ + ട = ണ്ട
  • ന + ത = ന്ത
  • മ + പ = മ്പ

    • ചന്ദ്രക്കലയുപയോഗിച്ച് കൂട്ടക്ഷരമുണ്ടാക്കുമ്പോൾ വിഭജിച്ച് എഴുതുന്നതിന് "്" double tap ചെയ്യുക.
  • സ + പ + "്" double tap + ത + ം = സപ്‌തം

    • പുതിയ വാക്ക് അല്ലെങ്കിൽ വാചകം സ്വരാക്ഷരങ്ങളിൽ തുടങ്ങുകയാണെങ്കിൽ shift key tap ചെയ്യേണ്ട ആവശ്യമില്ല. This option can be disabled using settings key in keyboard.
