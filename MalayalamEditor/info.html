Ez Malayalam Keyboard v1.0

A custom keyboard to write in Malayalam

With this custom keyboard, you can directly type Malayalam in any app.

Application Developed by:

Jijo Pulikkottil (jpulikkottil@gmail.com)

This application uses source code from Tasty Imitation Keyboard Project (https://github.com/archagon/tasty-imitation-keyboard) to implement custom keyboard. Thanks to that team for making such a great keyboard layout.
